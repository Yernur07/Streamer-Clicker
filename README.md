# Streamer Clicker — Game Project (Pygame)

### Project Description
A 2D Idle/Clicker game built using Python and Pygame where players simulate the life of a professional live streamer. Click the streamer to generate coins, purchase equipment upgrades, manage a real-time chat simulation, buy expensive weapon skins in the Marketplace, and hold giveaways to boost subscriber counts.

### Key Features
* **Functional Game Loop:** Interactive event handling, data updates, and continuous rendering at 60 FPS in Fullscreen mode.
* **Live Chat Simulation:** Dynamic pixel-art chat component powered by Python generators utilizing custom word wrapping.
* **Giveaway System:** Interactive "Guess the Number" sub-game where chat viewers try to guess a secret number to win skins.
* **Data Persistence:** Progress automatically saves and loads using a local JSON file.
* **Full OOP Architecture:** Complete modular object-oriented management for game items, progress tracking, and interactive bounds.

### Technologies Used
* **Language:** Python 3.x
* **Libraries:** Pygame, JSON, Random, Sys
